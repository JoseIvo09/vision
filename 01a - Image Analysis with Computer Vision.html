Image Analysis with the Computer Vision Service

Computer Vision é um ramo da inteligência artificial (IA) que explora o desenvolvimento de sistemas de IA que podem “ver” o mundo, seja em tempo real por meio de uma câmera ou analisando imagens e vídeos. Isso é possível porque imagens digitais são essencialmente apenas arrays de valores numéricos de pixels, e podemos usar esses valores como features para treinar modelos de machine learning que classificam imagens, detectam objetos e até geram descrições de fotos.

Use o Serviço Cognitivo de Visão Computacional

O Microsoft Azure inclui vários serviços cognitivos que encapsulam funções comuns de IA, incluindo alguns que ajudam a construir soluções de visão computacional.

O serviço cognitivo Computer Vision é um bom ponto de partida para nossa exploração de visão computacional no Azure. Ele usa modelos pré-treinados para analisar imagens e extrair informações.

Por exemplo, suponha que a Northwind Traders deseje implementar uma “loja inteligente”, na qual serviços de IA monitoram a loja para identificar clientes que precisam de ajuda e direcionam funcionários para atendê-los. Usando o serviço Computer Vision, imagens capturadas por câmeras podem ser analisadas para fornecer descrições úteis do que está sendo retratado.

Criar um recurso de Serviços Cognitivos

Vamos começar criando um recurso de Serviços Cognitivos na sua assinatura Azure:

  1. Em outra aba do navegador, abra o portal do Azure: https://portal.azure.com e faça login com sua conta Microsoft.

  2. Clique em ＋Criar um recurso , procure por Cognitive Services e crie um recurso com as seguintes configurações:

    • Nome : Informe um nome exclusivo
    • Assinatura : Sua assinatura Azure
    • Localização : Escolha uma região disponível
    • Camada de preços : S0
    • Grupo de recursos : Crie um grupo com nome exclusivo

  3. Aguarde a conclusão da implantação. Vá para o recurso e, na página Visão geral , clique no link para gerenciar as chaves do serviço. Você precisará do endpoint e das chaves para conectar seu aplicativo cliente ao recurso.

Obter a Chave e o Endpoint do recurso

Para usar o recurso de serviços cognitivos, as aplicações precisam do endpoint e da chave de autenticação :

  1. No portal do Azure, vá para a página Chaves e Endpoint do recurso e copie a Chave1 , substituindo YOUR_COG_KEY no código abaixo.
  2. Copie o endpoint do recurso e substitua YOUR_COG_ENDPOINT .
  3. Execute o código abaixo selecionando a célula e clicando no botão Executar célula (▶) à esquerda da célula. 
  cog_key = 'YOUR_COG_KEY'
cog_endpoint = 'YOUR_COG_ENDPOINT'

print('Ready to use cognitive services at {} using key {}'.format(cog_endpoint, cog_key))

Agora que configurou a chave e o endpoint, você pode usar o serviço para analisar uma imagem.

Para isso no Python, instale o pacote: 

  !pip install azure-cognitiveservices-vision-computervision

Agora você está pronto!

Execute o código abaixo para obter uma descrição de uma imagem localizada em /data/vision/store_cam1.jpg : 

  from azure.cognitiveservices.vision.computervision import ComputerVisionClient
from msrest.authentication import CognitiveServicesCredentials
from python_code import vision
import os
%matplotlib inline

# Caminho da imagem
image_path = os.path.join('data', 'vision', 'store_cam1.jpg')

# Cliente do serviço
computervision_client = ComputerVisionClient(cog_endpoint, CognitiveServicesCredentials(cog_key))

# Obter descrição
image_stream = open(image_path, "rb")
description = computervision_client.describe_image_in_stream(image_stream)

# Mostrar imagem e legenda
vision.show_image_caption(image_path, description)

Isso parece razoavelmente preciso.

Vamos tentar com outra imagem: 

  # Caminho da nova imagem
image_path = os.path.join('data', 'vision', 'store_cam2.jpg')

image_stream = open(image_path, "rb")

Mais uma vez, a legenda sugerida parece bastante precisa.

Analisar características da imagem

Até agora, você usou o serviço para gerar legendas descritivas. Mas ele pode fazer muito mais. Pode, por exemplo:

  • Detectar objetos comuns e suas localizações
  • Detectar rostos e estimar idade
  • Detectar conteúdo adulto, sugestivo ou violento
  • Gerar tags para facilitar buscas

Execute o código abaixo para analisar uma imagem de um cliente: 

  # Caminho da imagem
image_path = os.path.join('data', 'vision', 'store_cam1.jpg')

# Características a serem analisadas
features = ['Description', 'Tags', 'Adult', 'Objects', 'Faces']

# Analisar imagem
image_stream = open(image_path, "rb")
analysis = computervision_client.analyze_image_in_stream(image_stream, visual_features=features)

# Mostrar resultados
vision.show_image_analysis(image_path, analysis)
